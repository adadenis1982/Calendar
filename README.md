# Calendar

Используется:
* JS
* React.js
* Node.js

Использовать:
* nvm use 17
* npm i
* && npm start чтобы запустить приложение

Реализовано:
* верстка
* добавление/редактирование/удаление событий
* переход по месяцам
* сохранение календаря в localStorage
